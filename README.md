# 撸毛脚本集合

这个仓库包含了一系列实用的自动化脚本，用于简化各种任务和操作。

## 脚本列表

### BITZ 自动化部署脚本
- 功能：自动化安装Rust、Solana工具链，创建Solana钱包并部署bitz应用
- 适用系统：Linux/Unix
- 主要特点：
  - 自动安装所需依赖
  - 安全的密码输入处理
  - 自动配置环境变量
  - 钱包创建和管理

## 使用说明

每个脚本都包含详细的注释和使用说明。请在使用前仔细阅读脚本开头的说明部分。

## 注意事项

- 部分脚本可能需要root/sudo权限
- 请在使用前备份重要数据
- 建议在测试环境中先进行测试
